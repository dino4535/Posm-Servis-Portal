# POSM Backend API

Teknik Servis Portalı Backend API servisi.

## Kurulum

```bash
npm install
```

## Geliştirme

```bash
npm run dev
```

## Production Build

```bash
npm run build
npm start
```

## Environment Variables

`.env` dosyasını `.env.example` dosyasından kopyalayın ve gerekli değerleri doldurun.
